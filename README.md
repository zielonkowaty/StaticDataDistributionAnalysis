<h2>How to Run</h2>

<p>Follow the steps below to run the project locally:</p>

<h3>1. Download the Dataset</h3>

<p>
Download the CSV files from Kaggle using the link below:
</p>

<p>
<a href="https://www.kaggle.com/datasets/rockyt07/social-media-user-analysis">
https://www.kaggle.com/datasets/rockyt07/social-media-user-analysis
</a>
</p>

<p>
After downloading, extract the files and place them into the project folder.
Then update the file paths for <b>file</b> and <b>file1</b> variables in the code.
</p>

<hr>

<h3>2. Install Required Python Packages</h3>

<p>
Make sure you have <b>Python 3</b> installed.
</p>

<p>
Install all required dependencies using your IDE or the <code>pip</code> command.
</p>
